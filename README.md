# ASCIl-ART

Сайт для конвертации текста и изображений в ASCII/ASCIL ART.

## Возможности
- Конвертация текста в ASCII-арт.
- Конвертация изображений (`jpeg`, `png`, `webp`) в ASCII-арт.
- 4 режима генерации:
  - Детальный (расширенная таблица символов)
  - Стандартный (10 символов)
  - Блочный (символы блоков)
  - Бинарный (`0` и `1`)
- Выбор цвета ASCII-арта.
- Цветной ASCII-ART (каждый символ в цвет соответствующего пикселя исходного изображения).
- Скачивание результата в `.txt`.

## Запуск
Откройте `index.html` в браузере или поднимите простой сервер:

```bash
python3 -m http.server 8000
```

После этого откройте `http://localhost:8000`.
